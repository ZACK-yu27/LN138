# 词云生成工具规范文档

## 1. 项目概述

- **项目名称**: CloudWord Pro - 智能词云生成工具
- **项目类型**: Web应用程序 (Streamlit)
- **核心功能**: 支持中英文混合文本和Word文件上传的智能词云生成工具
- **目标用户**: 内容创作者、数据分析师、营销人员等需要文本可视化的人群

## 2. UI/UX 规范

### 2.1 布局结构

- **整体布局**: 左侧输入区(35%) + 右侧展示区(65%)的左右分栏布局
- **页面结构**:
  - 顶部: 标题栏(80px高)
  - 主体: 左右双栏内容区
  - 底部: 状态栏/版权信息(40px高)
- **响应式断点**:
  - 桌面端: > 1024px (左右分栏)
  - 平板端: 768px - 1024px (上下堆叠，缩放调整)
  - 移动端: < 768px (上下堆叠，全宽显示)

### 2.2 视觉设计

- **主色调**:
  - 主色: #6366F1 (Indigo 500)
  - 辅色: #8B5CF6 (Violet 500)
  - 强调色: #EC4899 (Pink 500)
  - 背景色: #0F172A (Slate 900) - 深色主题
  - 卡片背景: #1E293B (Slate 800)
  - 文字主色: #F8FAFC (Slate 50)
  - 文字辅色: #94A3B8 (Slate 400)

- **渐变方案**:
  - 标题区: linear-gradient(135deg, #6366F1 0%, #8B5CF6 50%, #EC4899 100%)
  - 按钮悬停: linear-gradient(135deg, #4F46E5 0%, #7C3AED 100%)

- **字体方案**:
  - 标题: "Noto Sans SC", "Segoe UI", sans-serif
  - 正文: "Noto Sans SC", "Segoe UI", sans-serif
  - 代码/数字: "JetBrains Mono", monospace
  - 标题字号: 32px (h1), 24px (h2), 18px (h3)
  - 正文字号: 16px, 辅助文字: 14px

- **间距系统**:
  - 基础单位: 8px
  - 卡片内边距: 24px
  - 区块间距: 32px
  - 元素间距: 16px

- **视觉效果**:
  - 卡片阴影: 0 4px 20px rgba(0, 0, 0, 0.3)
  - 边框圆角: 16px (大卡片), 12px (按钮), 8px (输入框)
  - 玻璃态效果: backdrop-filter: blur(10px); background: rgba(30, 41, 59, 0.8)

### 2.3 组件规范

#### 输入区域组件
1. **文本输入框**
   - 多行文本域，支持中英文
   - 占位符: "在此粘贴或输入文本..."
   - 高度: 300px (可拖拽调整)
   - 边框: 1px solid #334155, 聚焦时 #6366F1
   - 背景: #0F172A

2. **文件上传区**
   - 拖拽上传区域
   - 支持 .doc 和 .docx 格式
   - 显示文件列表和大小
   - 状态: 默认/悬停/上传中/已完成

3. **生成按钮**
   - 尺寸: 高度 48px, 宽度 100%
   - 背景: linear-gradient(135deg, #6366F1, #8B5CF6)
   - 悬停: 放大1.02, 阴影增强
   - 加载状态: 旋转图标 + "生成中..."

#### 展示区域组件
1. **词云图预览**
   - 响应式尺寸，最大宽度100%
   - 保持宽高比
   - 加载时显示骨架屏效果

2. **下载控制栏**
   - 格式选择: PNG / JPG 单选按钮组
   - 下载按钮: 带图标
   - 分辨率选择: 1x, 2x, 3x

#### 交互状态
- 按钮: default → hover (scale 1.02) → active (scale 0.98) → disabled (opacity 0.5)
- 输入框: default → focus (边框变色 + 发光) → error (红色边框)
- 卡片: 悬停时微微上浮 (translateY -2px)

## 3. 功能规范

### 3.1 核心功能

#### 3.1.1 文本输入
- 支持直接粘贴中英文混合文本
- 文本长度限制: 最大 100,000 字符
- 实时字符计数显示
- 清空文本按钮

#### 3.1.2 Word文件解析
- 支持 .doc 和 .docx 格式
- 最大文件大小: 5MB
- 解析失败时显示友好错误提示
- 同时支持文本输入和文件上传(合并内容)

#### 3.1.3 智能文本处理
- **中文分词**: 使用jieba库
- **英文分词**: 使用正则表达式，按空格和标点分割
- **停用词过滤**:
  - 中文停用词: 的、了、在、是、和、也、就、都、而、及、与、为、对、等、这、那、你、我、他、她、它、我们、你们、他们
  - 英文停用词: the, a, an, is, are, was, were, be, been, being, have, has, had, do, does, did, will, would, could, should, may, might, must, shall, can, need, dare, ought, used, to, of, in, for, on, with, at, by, from, as, into, through, during, before, after, above, below, between, under, again, further, then, once, here, there, when, where, why, how, all, each, every, both, few, more, most, other, some, such, no, nor, not, only, own, same, so, than, too, very
- **单字过滤**: 移除长度为1的词语
- **标点符号过滤**: 移除所有标点符号
- **数字过滤**: 可选移除纯数字

#### 3.1.4 词云生成
- 使用wordcloud库
- 支持自定义参数:
  - 词云形状: 矩形、圆形、心形、五角星
  - 颜色方案: 随机、彩色单色、渐变
  - 背景色: 可选透明或指定颜色
  - 最大词数: 200 (默认)
  - 字体: 使用HGLS.TTF支持中文

#### 3.1.5 图片导出
- 支持PNG和JPG格式
- 支持1x/2x/3x分辨率
- 文件名: wordcloud_时间戳.png/jpg

### 3.2 用户交互流程

1. 用户输入文本或上传Word文件
2. 点击"生成词云"按钮
3. 系统处理文本(分词、过滤)
4. 生成词云并显示在右侧
5. 用户选择下载格式和分辨率
6. 点击下载按钮保存图片

### 3.3 边界情况处理

- 空输入: 显示提示信息"请输入文本或上传文件"
- 文件格式错误: 显示错误提示，支持的文件类型
- 文件过大: 提示"文件超过5MB限制"
- 解析失败: 显示"文件解析失败，请检查格式"
- 无有效词语: 显示"未提取到有效词语，请检查输入内容"
- 生成超时: 30秒超时限制，显示超时提示

## 4. 技术栈

- **前端框架**: Streamlit
- **词云生成**: wordcloud
- **中文分词**: jieba
- **可视化**: matplotlib
- **Word解析**: python-docx
- **字体**: HGLS.TTF (项目提供)
