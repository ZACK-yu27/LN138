# 照片处理网页应用规范文档

## 1. 项目概述

- **项目名称**: Photo Processor
- **项目类型**: Python Web应用（Flask框架）
- **核心功能**: 提供基于Web的图片上传、处理和下载功能，支持黑白转换、缩放、翻转等基础图像处理
- **目标用户**: 需要快速处理图片的普通用户

## 2. UI/UX 规范

### 2.1 布局结构

```
+----------------------------------------------------------+
|                      页面标题                              |
+----------------------------------------------------------+
|  左侧区域 (40%)      |       右侧区域 (60%)               |
|  +----------------+  |  +------------------------------+  |
|  | 上传区域       |  |  | 处理结果展示区域              |  |
|  | - 点击/拖拽   |  |  | - 实时预览处理后图片          |  |
|  | - 文件选择    |  |  |                              |  |
|  +----------------+  |  +------------------------------+  |
|  +----------------+  |  +------------------------------+  |
|  | 预览缩略图    |  |  | 下载按钮                      |  |
|  +----------------+  |  +------------------------------+  |
+----------------------------------------------------------+
|                    控制按钮区 (左下角)                      |
|  [黑白照片]  [放大120%]  [缩小80%]  [左右互换]              |
+----------------------------------------------------------+
```

- **响应式断点**:
  - 移动端 (< 768px): 单列布局，左右区域堆叠
  - 平板 (768px - 1024px): 双列，比例50:50
  - 桌面 (> 1024px): 双列，比例40:60

### 2.2 视觉设计

- **主色调**:
  - 背景色: #f5f7fa (浅灰蓝)
  - 主色: #4a90d9 (柔和蓝)
  - 强调色: #67c23a (成功绿)
  - 文字色: #333333
  - 边框色: #dcdfe6

- **字体**:
  - 主字体: "Segoe UI", "Microsoft YaHei", sans-serif
  - 标题: 24px, font-weight: 600
  - 正文: 14px, font-weight: 400

- **间距**:
  - 容器内边距: 20px
  - 组件间距: 16px
  - 按钮内边距: 10px 20px

- **视觉效果**:
  - 卡片阴影: 0 2px 12px rgba(0, 0, 0, 0.1)
  - 按钮悬停: 背景色加深10%，过渡时间0.3s
  - 边框圆角: 8px

### 2.3 组件规范

- **上传区域**:
  - 虚线边框样式
  - 拖拽时边框变为实线+主色高亮
  - 支持文件类型: JPG, JPEG, PNG, GIF, BMP, WEBP

- **预览区域**:
  - 最大高度: 400px
  - 图片自适应缩放保持比例
  - 加载时显示Loading动画

- **处理按钮**:
  - 尺寸: 高度40px，宽度auto
  - 状态: normal / hover / active / disabled
  - 点击反馈: 轻微缩放效果

- **下载按钮**:
  - 尺寸: 宽度100%，高度50px
  - 强调样式: 主色背景，白色文字
  - 图标: 下载图标 + 文字

## 3. 功能规范

### 3.1 核心功能

1. **图片上传**
   - 支持点击选择文件
   - 支持拖拽上传
   - 支持格式: JPG, JPEG, PNG, GIF, BMP, WEBP
   - 文件大小限制: 10MB
   - 显示上传进度
   - 显示预览缩略图

2. **图片处理**
   - 黑白照片: 转换为灰度图像
   - 放大: 120% 尺寸
   - 缩小: 80% 尺寸
   - 左右互换: 水平翻转

3. **结果展示**
   - 实时显示处理后的图片
   - 与原图保持视觉对应
   - 支持重新处理

4. **图片下载**
   - 点击下载按钮保存处理后的图片
   - 默认文件名: processed_[原文件名]

### 3.2 API 端点

| 端点 | 方法 | 功能 |
|------|------|------|
| / | GET | 渲染主页面 |
| /upload | POST | 上传图片 |
| /process | POST | 处理图片 |
| /download/<filename> | GET | 下载处理后的图片 |

### 3.3 错误处理

- 不支持的文件类型: 提示"不支持的文件格式"
- 文件过大: 提示"文件大小超过10MB限制"
- 上传失败: 提示"上传失败，请重试"
- 处理失败: 提示"处理失败，请重试"

## 4. 技术实现

- **后端框架**: Flask
- **图片处理**: Pillow (PIL)
- **前端框架**: 原生HTML5 + JavaScript
- **临时文件存储**: 系统临时目录

## 5. 验收标准

1. 页面正确加载，无控制台错误
2. 可以通过点击和拖拽上传图片
3. 上传后显示预览缩略图
4. 点击处理按钮后右侧显示处理结果
5. 下载按钮可以保存处理后的图片
6. 响应式布局在不同设备上正常显示
7. 错误提示信息正确显示
