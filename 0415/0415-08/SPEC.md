# 数据探索应用规格说明书

## 1. 项目概述

- **项目名称**: DataExplorer - 交互式数据探索工具
- **项目类型**: Web应用程序（基于Streamlit）
- **核心功能**: 允许用户上传Excel/CSV文件并进行全面的数据探索、可视化和统计分析
- **目标用户**: 数据分析师、研究人员、业务人员

## 2. 技术栈

- **后端框架**: Streamlit 1.35+
- **数据处理**: pandas, numpy
- **可视化**: plotly, seaborn, matplotlib
- **文件处理**: openpyxl (Excel支持), xlrd (旧版Excel支持)

## 3. UI/UX 设计规范

### 3.1 布局结构

- **顶部**: 应用标题 + 文件上传区域
- **侧边栏**: 功能导航菜单
- **主内容区**: 各项分析结果显示

### 3.2 视觉设计

- **主题**: 深色专业主题，使用Streamlit暗色模式
- **主色调**: #00CC96 (绿色系 - 数据/分析)
- **辅助色**: #636EFA (蓝色), #EF553B (红色), #AB63FA (紫色)
- **背景**: 深灰色 #0E1117
- **字体**: System UI / Sans-serif

### 3.3 响应式设计

- 支持桌面端和移动端
- 图表自适应屏幕宽度
- 表格支持水平/垂直滚动

## 4. 功能规范

### 4.1 文件上传

- **支持格式**: .xlsx, .xls, .csv
- **文件大小限制**: 200MB
- **格式验证**: 检查文件扩展名和MIME类型
- **进度反馈**: 显示上传进度条
- **错误处理**: 格式错误、文件损坏等提示

### 4.2 数据概览

- 总行数
- 总列数
- 各列数据类型分布（数值型、文本型、日期型、布尔型等）
- 内存占用（MB）
- 文件大小

### 4.3 数据预览

- 显示前10行数据
- 支持水平滚动查看所有列
- 支持垂直滚动查看更多行
- 显示列名和数据类型

### 4.4 缺失值分析

- 每列缺失值数量
- 每列缺失比例（%）
- 缺失值热力图可视化
- 缺失值矩阵图

### 4.5 描述性统计

- 数值型特征统计表：
  - 计数(count)
  - 均值(mean)
  - 标准差(std)
  - 最小值(min)
  - 25%分位数(Q1)
  - 50%分位数(中位数/Q2)
  - 75%分位数(Q3)
  - 最大值(max)

### 4.6 特征相关性分析

- 相关系数矩阵（Pearson相关）
- 相关性热力图
- 可交互查看具体相关系数

### 4.7 数值型特征分布

- 每个数值型特征的直方图
- 核密度估计(KDE)图
- 支持选择特定特征查看

### 4.8 分类型特征分布

- 频数分布表
- 条形图可视化
- 支持选择特定特征查看

### 4.9 高级分析功能

#### 分组统计 (GroupBy)
- 选择分组字段
- 选择聚合列
- 选择聚合方式（sum, mean, count, min, max, std等）
- 自定义多级分组

#### 透视表分析 (Pivot Table)
- 选择行字段
- 选择列字段
- 选择值字段和聚合方式
- 支持多种聚合函数

## 5. 验收标准

### 5.1 功能验收

- [ ] 可成功上传.xlsx, .xls, .csv文件
- [ ] 上传过程显示进度和格式验证
- [ ] 数据概览正确显示行数、列数、类型分布、内存占用
- [ ] 数据预览正确显示前10行
- [ ] 缺失值分析正确计算数量和比例
- [ ] 缺失值热力图正确渲染
- [ ] 描述性统计包含所有必需指标
- [ ] 相关性矩阵和热力图正确生成
- [ ] 数值型分布图（直方图+KDE）正确显示
- [ ] 分类型分布表和条形图正确显示
- [ ] GroupBy功能正常工作
- [ ] Pivot Table功能正常工作

### 5.2 性能要求

- 支持至少10万行数据分析
- 页面响应时间 < 2秒（分析完成后）
- 图表加载时间 < 3秒

### 5.3 用户体验

- 响应式设计
- 加载状态提示（Spinner）
- 错误提示清晰
- 图表可交互（缩放、悬停显示详情）
